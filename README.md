# Mini-Project
Objective: promote a self-sustained “learn by doing” model that motivates students to gain skills through practical work instead of monetary payment.
Designed and developed a MERN full-stack website with separate Admin and Student interfaces.
Admin module: allows task creation, submission review, and performance evaluation through a points-based reward system.
Student module: enables students to view, complete, and submit tasks to earn points.
Integrated a points-to-learning system, where accumulated points unlock access to 5 technical courses hosted on the platform.
